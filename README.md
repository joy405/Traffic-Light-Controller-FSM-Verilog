# Traffic Light Controller using Verilog (FSM)

A Traffic Light Controller implemented in Verilog using a Moore Finite State Machine (FSM). The design controls highway and country road traffic using sensor input and timed state transitions. Simulated and verified using Vivado with a complete testbench and waveform analysis.

## Features

* 5-state FSM design
* Highway and Country road control
* Sensor-based switching
* Implemented in Verilog
* Simulated in Vivado

## Files

* `TFC.v` → Traffic Light Controller design
* `TFC_testbench.v` → Simulation testbench
* `FSM_diagram.png` → FSM state diagram

## Tools Used

* Verilog HDL
* Vivado
